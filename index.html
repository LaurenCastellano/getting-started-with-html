What are we focusing on?

  • Procedural Thinking - Breaking it into steps. Involves creating perfectly clear and unambiguous
    instructions for a computer to follow
  • Abstract Thinking - Apples and oranges example. finding similarity, or as programmers would
    say, generality amongst seemingly different things
  • Systematic Debugging - Identify the problem, break it down to what caused it then fix it
  • Technological Empathy - Start thinking like a computer and empathize with what it will
    understand.

Basics of Web and HTML

  • HTML – (Hyper Text Markup Language)main doc type of the web
  • URLs – how you refer to docs on the web
  • HTTP - (Hyper Text Transfer Protocol) protocol that the web runs on

Major pieces of the web

  • Browser – a program that runs on your computer to display files found on the web
  • Internet - worlds largest computer network
  • HTTP– the main protocol of the web
  • Servers- Computers that host the files that make up the web

HTML

    Heart of the web. The text content is what you see and markup is what it looks like. It references
    other docs such as images or videos and can link to other pages.

  • HTML Markup

    • Tags
      • <name> Content </name>
      • The closing tag has a / in it.
      • The whole this is an element
      • Void tags don't have a closing tag
      • Attributes - are a property of an HTML element like the herf for a link and teh anchor> is the <a>

    • Images
      • <img>
      • <img src="url" alt="add discriptive text for the blind">
      • Doesn't need closing tag because it is a void'

    • Inline vs. Block
      • Inline happens within a line of text but a block creates an invisible box around your text

    High level HTML documents

    • <!Doctype HTML> - Don't really understand this yet but it's a thing...
    • <html> opening and closing the whole document
    • <head> - Meta data if you include Java or Css it would go here
      • <title> </title> - the title of the page
    • <body> </body> - The body of the page
    • </html> - close it out
